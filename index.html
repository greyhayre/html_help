Структура

Тут указаны основные правила оформления тегов и общая структура.

  1. Абсолютно весь код должен находиться между тегами <html> ... </html>
  2. Основное тело сайта идет сразу после <html> <body> ... </body>
  3. Далее указывается тег для указания того, что написано в вкладке браузера <title> ... </title>
  4. Основная структура кода:
    1. Голова или шапка сайта <header> ... </header>
    2. Основная часть <main> ... </main>
    3. Подвал <footer> ... </footer>
  5. Практически все теги имеют пару, и текст находится между ними, как и другие теги.

Пример оформления:

<html>
    <body>
        <title>Надпись на вкладке браузера</title>
        <header>
            ...Код для шапки сайта...
        </header>
        <main>
            ...Код основной части...
        </main>
        <footer>
            ...код для подвала сайта...
        </footer>
    </body>
</html>

HTML

Тут основные теги и что они делают.

  1. <h1>...<h6> - Размер и оформление текста
  2. <p> - Перенести на следующую строку
  3. <a> - Добавить ссылку или иные интерактивные элементы (можно и в остальные теги, но этот не влияет на оформление)
  4. <img src="ссылка на картинку"> - Добавить картинку
  5. <input type="text"> - Поле, куда можно что-то написать:
    1. <table> - Создать таблицу
    2. <tr> - Строка таблицы
    3. <td> - Ячейка таблицы
      4. Столбец 1, ячейка 1 Столбец 2, ячейка 1
      Столбец 1, ячейка 2 Столбец 2, ячейка 2 
      <table>
    <tr>
        <td>Столбец 1, ячейка 1</td>
        <td>Столбец 2, ячейка 1</td>
    </tr>
    <tr>
        <td>Столбец 1, ячейка 2</td>
        <td>Столбец 2, ячейка 2</td>
    </tr>
</table>

CSS

Как оформлять стили?

  1. Оформление стиля внутри HTML-кода создается дополнительным параметром style: 
    <тег style="параметр: нужное значение; параметр: нужное значение;">
  2. Оформление в отдельном CSS-файле: 
    тег {
    параметр: нужное значение;
    параметр: нужное значение; 
}

Дизайн

Несколько правил того, как сделать красиво:

  1. Используй не больше 5 цветов, а лучше найди готовую палитру.
  2. Оформляй сайт в едином стиле.
  3. Не пиши много сплошного текста, разбавляй его картинками или цветными элементами.
  4. Соблюдай структуру — это очень важно.
