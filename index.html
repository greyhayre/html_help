Структура

Тут указанны основные правила оформления тэгов и общая структура

  1. Абсолютно весь код должен находится между тэгами < html > ... < /html >
  2. Основное тело сайта идет сразу после html < body > ... < /body >
  3. Далее указывается тэг для указания того что написанно в вкладке браузера < title > ... < /title >
  4. Основная структура кода:
    1. Голова или шапка сайта < header > ... < /header >
    2. Основная часть < main > ... < /main >
    3. Подвал < footer > ... < /footer >
  5. Практически все тэги имееют пару и текст находится между ними как и другие теги

Пример оформления:

< html >
    < body >
        < title > Надпись на вкладке браузера < /title >
        < header >
            ...Код для шапки сайта...          
        < /header >
        < main >
            ...Код основной части...
        < /main >
        < footer >
            ...код для подвала сайта...
        < footer >
    < /body >
< /html >

HTML

Тут основные тэги и что они делают

  1. < h1 >...< h6 > - Размер и оформление текста
  2. < p > - перенести на следующую строку
  3. < a > - добавить ссылку или иные интерактивные элементы(можно и в остальные тэги но этот не влияет на оформление)
  4. < img src ="ссылка на картинку"> - Добавить картинку
  5. < input type = "text" > - Поле куда можно что то написать:
    1. < table > - создать табличку
    2. < tr > - столбец таблицы
    3. < td > - ячейка таблицы
      4. столбец 1 ячейка 1 столбец 2 ячейка 2
      столбец 2 ячейка 1 столбец 2 ячейка 2 
      < table >
    < tr >
        < td > столбец 1 ячейка 1 < /td >
        < td > столбец 2 ячейка 1 < /td >
    < /tr >
    < tr >
        < td > столбец 2 ячейка 1 < /td >

Какие есть доп значения для тэгов?

  1. что это за доп параметр? и где его писать 
    < тэг доп.параметр = "" >
  2. style = "..." - Задать стили
  3. href = "..." - Задать ссылку
  4. href="#hhh" - ссылка внутри нашего сайта
  5. id ="hhh" - тут указываем куда нас отправит сайт по ссылке выше
  6. pclass ="..." - Уникальный селектор
  7. src ="..." - Используется в тэге img чтобы задать ссылку на картинку

CSS

Как оформлять стили?

  1. Оформление стиля внтри html кода создается доп. параметром style = 
    < тэг style="параметр: нужное значение; 
параметр: нужное значение;" >
  2. Оформление в отдельном CSS файле "Селектор"
    3. тэг {
параметр:нужное значение;
параметр:нужное значение; 
}

Какие есть параметры и значения?

  1. color: rgb(255, 255, 255); - цвет текста
  2. background-color: rgb(255, 255, 255); - цвет фона
  3. border: 4px solid rgb(255, 255, 255); - цвет обводки и ее толщина
  4. text-align: center; - расположение текста
  5. font-size: 20px; - размер текста
  6. font-family: Arial; - Шрифт текста
  7. font-weight: bold; - Стиль теста(жирный курсив и тд)
  8. text-decoration: none; - отключить стандартное оформление(например подчеркивание ссылок)
  9. border-radius: 10px; - скругление углов обводки
  10. padding: 5px; - внутренние отступы
  11. margin:10px; - внешние отступы

Иерархия селекторов

  1. .тэг - Уникальный селектор
  2. тэг тэг - Контекстный селектор
  3. тэг - Селектор
  4. style = - Задание стиля внутри кода
  5. defaut - Стандартные настройки

Дизайн

Несколько правил того как сделать красиво

  1. Используй не больше 5 цветов а лучше найди готовую палитру
  2. Оформляй сайт в едином стиле
  3. Не пиши много сплошного текста разбавляй его картинками или цветными элементами
  4. Соблюжай структуру это очень важно
